# AI Research Assistant using Amazon Nova

## Overview
This project is an AI-powered Research Assistant that helps users extract information from large documents using natural language questions. It uses Amazon Nova foundation models and a Retrieval-Augmented Generation (RAG) pipeline to provide accurate and context-aware answers.

## Features
- Ask questions directly from documents
- Semantic search using vector embeddings
- Context-aware answers generated by Amazon Nova
- Interactive web interface for document upload and querying

## Tech Stack
- Python
- Amazon Nova 2 Lite
- Amazon Nova Multimodal Embeddings
- AWS Bedrock
- LangChain
- FAISS (Vector Database)
- Streamlit
- NumPy
- Pandas

## How It Works
1. Documents are uploaded and split into smaller chunks.
2. Each chunk is converted into embeddings using Amazon Nova Embeddings.
3. Embeddings are stored in a FAISS vector database.
4. When a user asks a question, the system retrieves relevant chunks.
5. Amazon Nova generates an answer using the retrieved context.
